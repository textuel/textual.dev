A brief introduction


Introducing Textuel, a team aimed with the sole goal of making the world a better place with the power of programming.

저희는 2019년에 프로그래밍을 좋아하는 두 학생이 하나의 목적을 위해 설립한 작은 팀입니다.

Skills

저희 팀에서는 다음과 같은 스택들을 다룹니다. 급변하는 웹 기술을 따라가기 위해 아낌없이 투자합니다.

Node.js

빠른 고성능 서버를 노드로 손쉽게 만들 수 있습니다

React

리엑트로 빠른 웹 애플리케이션 개발을 합니다

AWS

애플리케이션 배포는 안전하고 효율적인 AWS를 이용합니다

Express

Express를 이용해 효율적인 서버 개발을 합니다.

git

git을 이용하여 버전 관리를 확실히 합니다

Python

파이썬을 이용하여 데이터를 확보, 처리합니다.

SASS

SASS를 이용하여 손쉽게 스타일 관리를 합니다

Bootstrap

빠른 개발을 위해 부트스트랩을 이용합니다

Docker

최대한 같은 개발 환경을 이용하기 위해 도커를 사용합니다
